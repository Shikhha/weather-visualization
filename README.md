## Description

A fun weather visualization application built with react

## Installation

To install

```bash
npm install
```

To run

```bash
npm start
```

## API key

- Obtain a Google maps api key and paste in bootstrapURLKeys in GoogleMap.js

## Built With

- React.js
- Bootstrap4

## Packages used

- react-sparklines
- google-map-react
